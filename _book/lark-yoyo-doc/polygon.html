切换网络至Polygon

添加并切换到Polygon网络

Lark采用的是Polygon网络，所以在使用前需要配置网络参数并切换到Polygon网络（MetaMask中设置）

网络设置：

Network Name: Polygon

New RPC URL: https://polygon-rpc.com

ChainID: 137

Symbol: MATIC

Block Explorer URL: https://polygonscan.com
