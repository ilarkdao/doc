Lark简介

简介

Lark将是一系列智能合约的协议集，映射现实中的价值关系，以实现价值维度的元宇宙。整体将采用模块化、渐进式地设计， 以实现可拓展、可叠加的协议效果。它将包含这些模块： 去中心化社交YOYOW + DeFi + NFT + DAO等。它将在DAO的组织形式下进行协作，以实现自组织自进化自完成。
